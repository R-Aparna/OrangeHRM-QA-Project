# OrangeHRM-QA-Project
This repository contains a manual QA testing project focused on the
Login feature of the OrangeHRM demo web application.

The project demonstrates test planning, test case design,
defect identification, and validation of UI, usability,
and security-related behaviors.

## Features Tested
- Login with valid and invalid credentials
- Required field validations
- Password masking and paste behavior
- Username case sensitivity
- Error message handling
- Space character handling

## Tools & Skills Demonstrated
- Manual Testing
- Functional & Negative Testing
- Test Case Design
- Defect Logging
- Web Application Testing
