# OrangeHRM Login – QA Testing Project

## Project Overview
This project demonstrates manual QA testing of the **Login functionality** of the OrangeHRM demo web application.

The goal of this project is to showcase real-world QA skills including:
- Test planning
- Test case design
- Defect identification and analysis
- Attention to edge cases and usability behavior

## Application Under Test
**Application:** OrangeHRM Demo  
**Module:** Login  
**Type:** Web Application  

## Scope of Testing
- Login with valid and invalid credentials
- Required field validation
- Password masking
- Username case sensitivity
- Space character handling
- Error message behavior
- Post-error login behavior

## Testing Approach
- Manual functional testing
- Positive and negative testing
- Boundary and edge case validation
- UI and usability checks

- Designed detailed test cases covering positive, negative, and edge scenarios
- Identified functional and usability defects based on actual application behavior
- Applied severity and priority based on real-world impact
